# Password Checker Assignment

## Objective
Complete the `is_valid_password` function to ensure it checks for the following criteria
- At least 8 characters long
- At least one uppercase letter
- At least one lowercase letter
- At least one digit
- At least one special character

Instructions for Students
Clone the repository to your local machine.
Open the password_checker.py file.
Complete the is_valid_password function by adding the missing code.
Run the tests using python -m unittest discover __tests__.
Commit and push your changes to your GitHub repository.
